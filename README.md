# Instruct Pix2Pix Python Script

```
python3 main.py
```

## Requirements

* pytorch
* diffusers
* pillow

## Installation

1. Download the script
2. Run `pip3 install -r requirements.txt`

## Instructions

1. Run the script
2. Enter the URL of the .jpg or .png
3. Enter the prompt
4. Your result is in `result.png`
